# Threat Modeling Materials

本目录用于支持架构评审场景下的威胁建模落地，覆盖：立项、问卷、访谈、风险评估、整改复查与周期复评。

## 文件清单

## Excel 调研模板

- `TM-01-项目立项与范围确认表.xlsx`
  - 立项信息、评估范围、RACI、里程碑
- `TM-02-业务问卷与应用信息收集表.xlsx`
  - 业务基础问卷、应用信息收集、安全现状问卷
- `TM-03-多轮访谈记录与ToDo跟踪.xlsx`
  - 首次/二次访谈、分歧闭环、会后 ToDo
- `TM-04-威胁识别与风险评估台账.xlsx`
  - 资产定义、威胁识别、风险评估、缓解策略与排期
- `TM-05-风险处置与复查闭环表.xlsx`
  - 整改计划、复查验证、补偿监控、培训计划、周期复评

## 文档模板

- `01_威胁建模项目启动指南.md`
- `02_问卷设计与发放说明.md`
- `03_访谈执行SOP_40分钟.md`
- `04_威胁评估报告模板.md`
- `05_双人复核与风险处置准则.md`

## 使用建议（推荐节奏）

1. 先填写 `TM-01`，完成范围和资源投入确认。
2. 发放 `TM-02`，作为项目启动前置条件。
3. 使用 `TM-03` 进行首次/二次访谈闭环。
4. 在 `TM-04` 中形成正式威胁与风险台账。
5. 在 `TM-05` 中跟踪整改、复查、监控补偿与复评计划。

## 方法原则

- 增量系统尽量全覆盖，存量系统按攻防优先级分批推进。
- 业务与安全共同参与，避免“你问我答”式对立氛围。
- 风险对齐聚焦事实与止损，目标是长期改善安全状况。
- 发现风险容易，推动解决并验证闭环才有价值。
