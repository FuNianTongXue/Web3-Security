# vuln-suite（静态扫描测试样本）

该样本用于验证静态审计能力，**禁止用于生产环境**。

## 包含漏洞点（示例）

1. `tx.origin` 权限认证
2. 重入漏洞（先调用后记账）
3. 循环外部调用导致 DoS 风险
4. 忽略 `call` 返回值
5. 未检查 ERC20 返回值
6. 可预测随机数（时间戳/区块）
7. 用户可控 `delegatecall`
8. 弱访问控制修改逻辑地址
9. 时间戳依赖
10. 任意白名单可 `selfdestruct`
11. 不安全 assembly 读取
12. 硬编码弱口令哈希

## 推荐测试方式

- 在你的平台中选择：
  - 本地目录扫描：导入 `vuln-suite` 目录
  - 压缩项目扫描：导入 `vuln-suite.zip`
